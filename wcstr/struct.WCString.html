fn split(self, delimiter: u16) -> Split

Split the string into multiple &mut WCStr using a delimiter.

  • This returns an iterator that creates a &mut WCStr for each part of the string separated by the delimiter.
  • This will consume the string.

split() example

use wcstr::WCString;
let s = WCString::from_str("a;b;c;d;e").unwrap();
let mut count = 0;
for w in s.split(b';' as u16).iter() {
    count += 1;
    assert!(w.len() == 1);
}
assert!(count == 5);
